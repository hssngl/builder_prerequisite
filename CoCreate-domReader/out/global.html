domEditor(paramObject)

domEditor used to run a dom command in remote and now used to do all sort of things
Parameters:
Name Type Description
paramObject param objects
Properties
Name Type Attributes Default Description
context element <optional>
 document the context in which the selector_type and selector is called to resolve to a/list of target
target element | Array.<element> <optional>
 alternatively you can feed the target by setting target. if you set target, selector_type, selector and context is skiped.
index integer <optional>
 null if the index specified and target be an array, it will select that target in that array
selector_type string <optional>
 selector_type will be called on context, example: querySelectorAll, querySelector, getElementById,
selector string <optional>
 is the parameter of the selector_type and applied to it, example: #id_example, .class
method string <optional>
 null the method that will be called on targets
property string <optional>
 css property can alternatively define here. example: null,length,color
value string | array | object <optional>
 if method param is defined, value is applied as a parameter, example: “hola, hey” ; “hola”, “hey”; “string”; {‘first_attr':'cvalue','second_attr':'value'}, ['positon', 'value'] if you give an array it is going to invoke method for every element in the array
